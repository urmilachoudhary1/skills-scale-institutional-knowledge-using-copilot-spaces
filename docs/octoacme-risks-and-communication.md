## Communication and Escalation Matrix

| Situation                      | First Contact      | Secondary Contact      | Escalation Point           |
|--------------------------------|--------------------|-----------------------|----------------------------|
| Blocker in daily work          | Developer/QA       | Project Manager        | Product Manager / Sponsor  |
| Requirement unclear            | Business Analyst   | Product Manager        | Project Manager            |
| Critical prod incident         | DevOps/Engineer    | Project Manager        | Sponsor / Incident Lead    |
| Customer escalation/major bug  | Customer Support   | Product Manager        | Project Manager            |
| User experience feedback       | UX Designer        | Product Manager        | Project Manager            |

- For all major risks, ensure the risk register is updated and team is informed during sync meetings.
- Use documented templates for stakeholder or incident comms when appropriate.
